Tier 1 Documentation

This section outlines how to use the Tier 1 tools within Moleculogic for MDC logic validation, simulation, and blueprint refinement.

1. Blueprint Explorer

The explorer lets you scan through existing molecular logic blueprints. It shows the title, coordination type, logic conditions, and status tags.

2. Ligand Simulator

This tool allows interactive input of ligand coordination, pH range, reactivity logic, and outputs simulation feedback such as binding class, entropy, and instability warnings.

3. Saving and Promoting Blueprints

Validated blueprints can be saved for future retrieval and comparison. Those scoring above threshold (Tier 2-Ready) can be promoted with admin or system verification.

4. Access Control

Tier 1 features are NDA-gated. Access control is based on email-linked tokens validated via the NDA handler system. If a user is not approved, simulation is disabled.

5. FDC Compatibility

While Tier 1 is currently focused on MDC, selected functions will soon support FDC blueprint logic (ligands, peptide chains, synthetic proteins). Documentation will evolve accordingly.
