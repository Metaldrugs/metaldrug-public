The SEI-Lite engine provides automated scoring for each submitted FDC logic set using the following three dimensions:

Entropy Class

  • Class A: Stable, convergent logic with minimal redundancy
  • Class B: Moderately redundant but interpretable logic patterns
  • Class C: Highly disordered, likely non-functional without revision

Confidence Score

Numerical value (0-1.0) reflecting the predictive reliability of the logic path based on SEIâ€™s simulation models and dataset priors.

Functional Risk Level

  • Low: Demonstrates coherent interaction modeling and functional outputs
  • Moderate: Partial coherence, risks of side effects or off-target actions
  • High: Significant instability or biologically implausible constructs

These metrics enable researchers and reviewers to rapidly prioritize, revise, or escalate FDC proposals through the Moleculogic pipeline.
