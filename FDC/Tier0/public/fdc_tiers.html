Tier 0 - Public Preview

Open access to general overview pages, use case previews, and the licensing interest submission form. No sensitive logic or compound identifiers are shared.

Tier 1 - Review Access

Requires NDA. Includes access to FDC whitepapers, reviewed applications, non-public Moleculogic simulations, and selected data logs.

Tier 2 - Logic Customization

Invite-only. Grants the ability to test logic permutations using Moleculogic Blueprint Builder and evaluate API-linked FDC logic performance metrics.

Tier 3 - Secure Deployment

Highest security level. Allows for full integration of FDC logic into proprietary applications with encrypted licensing and real-time feedback channels.
